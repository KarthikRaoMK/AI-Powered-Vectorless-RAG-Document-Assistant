\# AI-Powered Vectorless RAG Document Assistant



A full-stack AI-powered document question-answering system built using Spring Boot, Spring AI, React, MySQL, and Google Gemini.



\## Tech Stack



\- Java 21

\- Spring Boot

\- Spring AI

\- React (Vite)

\- MySQL

\- Google Gemini

\- Apache PDFBox



\## Features



\- PDF Upload

\- Intelligent Text Chunking

\- Vectorless RAG

\- AI Question Answering

\- Source Citation

\- Document Management
